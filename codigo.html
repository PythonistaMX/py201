Bienvenidos pythonistas

Un espacio para los entusiastas de Python de habla hispana.

 

Python es uno de los lenguajes de programación  más populares actualmente.

Al ser un lenguaje de propósito general, puede ser utilizado en prácticamente cualquier tipo de aplicación imaginable. Su versatilidad permite ejecutar código desde un modesto dispositivo de IoT, hasta en un cluster conformado por cientos de servidores. 


  

 

El término "Pythonista" se refiere a aquel programador que es eficaz en el uso de las particularidades de Python.

¿Qué hacemos? 

Pythonista® es un esfuerzo para fortalecer a la cada vez más extensa comunidad de pythonistas novatos y expertos de habla hispana mediante la realización de:

  • Eventos, tales como los Meetups que se realizan periódicamente en la Ciudad de México.
  • Capacitación presencial y en línea.
  • Apoyo a las comunidades hermanas de pythonistas y programadores en general.
  • Publicación de ofertas de empleo.
  • Apoyamos a las escuelas ofreciendo cursos a precios preferenciales e incluso gratuitos a grupos de alumnos.

Esta es tu comunidad y estamos felices de darte la bienvenida.

Nuestro curso de Python 3 desde cero.

Si acabas de acercarte a Python, te recomendamos que explores nuestro Curso de Introducción a Python en donde encontrarás recursos de aprendizaje junto con apuntes interactivos, los cuales consisten en notebooks de Jupyter que incluyen los apuntes con el código ejecutable.

Accede mediante Microsoft Azure Notebooks.

Hemos puesto a tu disposición todas las notebooks del curso en esta dirección. Si tienes una cuenta de Microsoft (Hotmail, Outlook, etc.), puedes acceder y ejecutar las notebooks totalmente en línea.

Descarga Anaconda.

Jupyter es un servidor de notebooks interactivas que puede ser desplegado mediante un servidor web local. Si no eres usuario experto, te recomendamos descargar e instalar Anaconda, una distribución de Python fácil de instalar tanto en Windows como en MacOS X y Linux, la cual incluye entre otras cosas un servidor de Jupyter. Los apuntes fueron diseñados para Python 3, así que es importante que descargues la versión para Python 3.6 o superior.

Nuestras máquinas virtuales.

Existe la problemática de que los entornos para la ejecución de las notebooks de los cursos Pythonista® son muy heterogéneas. Es por eso por lo que hemos publicado máquinas virtuales compatibles con Virtualbox, las cuales incluyen un servidor Jupyter habilitado así como los contenidos más actualizados de nuestros cursos. Las máquinas virtuales están configuradas de tal forma que sólo es necesario iniciarla y el resto de las operaciones se realizan desde el navegador de tu sistema de escritorio.

Puedes ver nuestras máquinas virtuales en el apartado de descargas.
